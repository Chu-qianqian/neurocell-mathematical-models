# How to read the equation pages

The LaTeX on these pages describes **conceptual mathematical structures** of model families. It is neither a verbatim transcription of copyrighted papers nor equation extraction for the current seed records. A catalogue entry becomes `extracted` only after recording source location, variables, parameters, and an independent verification step.

The notation below is therefore for readability. Users must return to the original papers for complete equations, parameters, and units.
