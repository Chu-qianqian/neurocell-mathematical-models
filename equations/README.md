# 方程页面的阅读规则

这里的 LaTeX 用于说明模型家族的**概念性数学结构**。它们不是对受版权保护论文的逐式复制，也不是现有种子记录的方程提取。只有记录了公式位置、变量、参数和独立核验的信息后，模型目录的 `equation_status` 才会改为 `extracted`。

因此，下面页面中的符号仅服务于可读性：用户必须回到原始论文核对完整公式、参数和单位。
