# Astrocytes

G-ChI is the current bibliographically verified seed. The Postnov record is classified as a neuron–astrocyte coupled model, not an astrocyte single-cell model. De Young–Keizer is in the candidate screening queue and must not be treated as a complete astrocyte model before its equations and scope are verified.

Priority topics include intracellular Ca²⁺, IP3, CICR, gliotransmitters, tripartite synapses, Ca²⁺ waves, gap-junction/IP3 diffusion, K⁺ buffering, glutamate uptake, Na⁺/K⁺ homeostasis, metabolic coupling, cell volume, and neurovascular coupling. Each source must distinguish single-cell modules, networks, tissue-scale models, and vascular coupling.
