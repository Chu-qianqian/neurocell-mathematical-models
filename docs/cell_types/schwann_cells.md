# Schwann cells

This core release contains no Schwann-cell record that meets the joint gate of clear source, explicit equation, and clear cellular scope. This is not a claim that the field has no models. Later searches will cover peripheral-nerve myelination, differentiation, axon–Schwann-cell interactions, injury-induced dedifferentiation, repair Schwann cells, Wallerian degeneration, regeneration, and remyelination.

Candidates must state whether they model the Schwann cell itself, myelin/conduction parameters, injury/regeneration at tissue scale, or an indirect representation at another level.
