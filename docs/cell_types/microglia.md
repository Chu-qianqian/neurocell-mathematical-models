# Microglia

The [Amato and Arnold record](../../equations/models/amato_arnold_2025_microglia.md) is bibliography verified for ischemic-penumbra population dynamics. Its equations, variables, parameters, and code license have not yet been audited, so no microglial equation is displayed.

Priority screening topics include chemotaxis, migration, phagocytosis, purinergic signalling, inflammatory feedback, synaptic pruning, and disease-state population models. Original source terminology is retained with its stated scope and limitations.
