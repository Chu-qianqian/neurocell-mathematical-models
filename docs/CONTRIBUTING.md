# Contributing

Please submit changes through a branch and pull request. Do not add paper PDFs, publisher figures, supplements, or code unless you document an explicit reuse license and provenance.

For a new model record, include the primary persistent identifier, target cell type, modelling formalism, evidence for the classification, and independent statuses for bibliography, equations, and implementation. Run `python scripts/validate_catalogue.py` before submitting.

Use `candidate` when in doubt; promotion to `verified_bibliography` needs a metadata check against an authoritative source.
