# Equation notation policy

Displayed mathematics uses GitHub Markdown LaTeX. A source-specific page identifies each system as either an exact verified transcription or an equivalent notation with a documented mapping. Conceptual schematics are not stored under source-specific model filenames.

Symbols retain the meanings assigned by the cited source whenever possible. When a conventional rendering resolves a typography or sign-convention issue, the page states the mapping and does not claim a verbatim transcription. Units, parameter values, and numerical details are marked `not verified` or `parameters_incomplete` until they are individually audited.

The repository writes explanatory prose independently. Equations are cited to their original sources and are limited to the mathematical system needed to identify the model.
